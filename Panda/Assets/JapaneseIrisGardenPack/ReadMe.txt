By using this package, you can make the Japanese "iris garden".

The package contains:

-16 object types(fences, lamps, etc...) and some variations.
-46 plant variations.
-100 rock variations.
-4 fish(japanese nishikigoi) variations and include 4 animations.
-5 tree types and 11 variations.

Update:
The scale of meshes was adjusted.
